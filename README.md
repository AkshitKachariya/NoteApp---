# 📝 Keep Note App

A modern and responsive **Note Taking / Task Management App** built using **React, Vite, and Tailwind CSS**.

This app allows users to create, manage, and delete notes with a clean UI, dark mode support, and local storage persistence.

---

## 🚀 Live Demo

🔗 Live Website: [https://note-app-akshit1.vercel.app/](https://note-app-akshit1.vercel.app/)

---

# ✨ Features

* ✅ Add Notes & Tasks
* ✅ Delete Notes
* ✅ Modern Responsive UI
* ✅ Dark / Light Theme Toggle
* ✅ Local Storage Support
* ✅ Smooth Scroll to Top Button
* ✅ Mobile Friendly Design
* ✅ Built with React + Tailwind CSS

---

# 🛠️ Tech Stack

* **React.js**
* **Vite**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **LocalStorage API**
